# Git Practice Repository

This repository is created to practice Git and GitHub basics daily.

## What I know so far

- Git is a version control system
- GitHub allows multiple developers to collaborate on a single project.
- Git helps track changes in files
- Git allows going back to previous versions
- GitHub is used to store code online

## Commands I practice

- git init
- git status
- git clone 
- git add
- git commit
- git push
- git pull
- git remote
- git branch
- git diff
- git merge
- git log
- git checkout / git switch
- git reset

## Goal

Build consistency with Git workflow and commit history.
This repository focuses on consistency and Git workflow rather than project size.
